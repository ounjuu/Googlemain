## :hammer: 주요기능
#### :star: HTML과 CSS만으로 구글 메인 페이지 구현<br/>
1. HTML: div, class, img 태그를 사용하였습니다.
2. CSS: display, margin, position, border-radius, z-index, hover 등을 사용하였습니다.
3. a 태그를 사용하여 클릭 시 실제 페이지로 이동하게 구현하였습니다.
